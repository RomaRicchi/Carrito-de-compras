📦 CAPSTONE – Tienda Online de Gorras

CAPSTONE es una aplicación web desarrollada con React + TypeScript que simula una tienda online de gorras con carrito de compras.
El objetivo del proyecto es aplicar buenas prácticas de desarrollo frontend, tipado estricto y manejo de estado global sin depender de un backend.

🛠️ Tecnologías utilizadas

React – Librería principal para la construcción de la interfaz

TypeScript – Tipado estático para mayor seguridad y mantenibilidad

Vite – Herramienta de desarrollo y bundling

Bootstrap – Estilos base y sistema de grillas

Context / Custom Hooks – Manejo del estado global del carrito

JavaScript ES6+ – Sintaxis moderna

HTML5 & CSS3

🧱 Estructura del proyecto

Componentes reutilizables

Header: navegación y visualización del carrito

CapItem: tarjeta de producto

Datos mockeados

Listado de gorras cargado desde un archivo local (db.ts)

Estado del carrito

Agregar productos

Incrementar / decrementar cantidades

Eliminar productos

Vaciar carrito

Cálculo automático del total

🛒 Funcionalidades principales

Visualización de una colección de gorras

Agregado de productos al carrito

Control de cantidades por producto

Eliminación individual de productos

Vaciado completo del carrito

Cálculo dinámico del total a pagar

Interfaz responsive

🎯 Objetivo del proyecto

Este proyecto fue desarrollado como práctica académica para reforzar conceptos de:

Componentización en React

Uso de hooks personalizados

Manejo de estado global sin Redux

Tipado correcto con TypeScript

Separación de responsabilidades

Persistencia del carrito en localStorage


🚀 Posibles mejoras futuras

Integración con backend y base de datos

Autenticación de usuarios

Pasarela de pagos

Filtros y búsqueda de productos
